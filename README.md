## 🔑 STEP 1: Firebase Keys Lagao

`src/firebase.js` file kholein aur apni keys paste karein:

```js
const firebaseConfig = {
  apiKey:            "APNI_API_KEY",       // <- Yahan apni key
  authDomain:        "APNA_AUTH_DOMAIN",   // <- Yahan
  projectId:         "roz-kamao-pk",       // <- Yahan
  storageBucket:     "APNA_BUCKET",        // <- Yahan
  messagingSenderId: "APNA_SENDER_ID",     // <- Yahan
  appId:             "APNA_APP_ID",        // <- Yahan
};
```

---

## 🚀 STEP 2: Vercel Pe Deploy Karein

1. vercel.com pe login karein
2. "Add New Project" click karein
3. "Import Git Repository" — ya directly folder upload karein
4. Framework: **Create React App** select karein
5. Deploy click karein!
